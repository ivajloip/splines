* Необходими инструменти за компилиране на кода:
  - Под linux: qt4-dev, qmake-qt4, g++
  - Под windows: qt-sdk, qt-creator (поне прави нещата по-лесни, принципно може и без него)

* Компилиране на кода (под Linux):
  1. qmake -project
	2. qmake 
	3. make

* Възможности:
  - Ctrl + O -> отваря диалог за избор на файл за четене.
  - Ctrl + S -> отваря диалог за избор на интересуващите ни точки и след това 
    диалог за избор на файл за записване на пресметнатите стойности. Не е
    разрешен, ако няма предварително прочетен файл.
  - Esc -> излизане от програмата

* Идеи за подобрение:
  - онагледяване на пресметнатата функция (в рамките на няколко дни би трябвало
      да успея да го направя)
  - Графично въвеждане на данни в таблица (отново би трябвало да може да стане
      в рамките на седмица)
  - Радактириране на данните директно в мястото, където се онагледява пресметнатата
    функция (по-сложно)
 
* Инсталиране под windows - разархивиране на архива splines.rar
